QuickQuiz

An interactive trivia game built with plain HTML, CSS, and JavaScript — no frameworks or dependencies.

Live Demo: rahularyann1.github.io/quickquiz-game

Features


3 Categories – Computer Science, Cybersecurity, and General Knowledge
Timed Questions – 15-second countdown per question with a visual progress bar
Dynamic Scoring – base points for correct answers plus a speed bonus for fast responses
Streak Tracking – tracks current and best answer streaks
Live Scoreboard – terminal-style header showing score, question count, and streak in real time
Results Summary – final score, accuracy, best streak, and average response time
Replay – play again or switch categories instantly


How to Run

Option 1: Open directly

Just open index.html in any modern browser — no setup required.

Option 2: Host on GitHub Pages


Upload index.html to a GitHub repository (root folder)
Go to Settings → Pages
Set source to the main branch, / (root) folder
Visit https://yourusername.github.io/repo-name/


Tech Stack


HTML5
CSS3 (custom properties, flexbox, grid, animations)
Vanilla JavaScript (DOM manipulation, timers, no libraries)
